# Enhanced Passive Tree Analysis - Raw Idea

## Feature Description

Enhanced Passive Tree Analysis - Enable the AI to parse, understand, and analyze Path of Building passive skill trees. This includes:
- Parsing passive tree data from PoB XML files
- Identifying allocated nodes, keystones, and notable passives
- Understanding build archetypes based on tree allocation patterns
- Providing tree optimization suggestions
- Analyzing pathing efficiency and point allocation

## Project Context

- **Product:** Exile's AI Companion (MCP server for PoB analysis)
- **Priority:** Item #1 on the roadmap (Medium - 1 week estimated)
- **Current state:** Phase 0 and Phase 1 complete (basic build reading + file watching)
- **Tech stack:** TypeScript/Node.js, MCP SDK, fast-xml-parser
